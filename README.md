# 🐝 Smart Bees — Marketing Analytics Demo

An AI-powered marketing analytics tool built with Python and the Claude API.

## Features
- Upload any CSV dataset
- Auto-detect business KPIs
- Interactive charts
- Streaming AI insight report in French
- Data quality checks

## Tech Stack
- Python
- Streamlit
- Claude API (Anthropic)
- Plotly
- Pandas

## Setup
```bash
pip install streamlit anthropic pandas plotly
export ANTHROPIC_API_KEY="your-key-here"
streamlit run app.py
```
